# La Divergente — Starter (Astro + Netlify CMS)

Este starter es un punto de partida para publicar en Netlify y editar contenidos desde /admin.

## Requisitos
- Cuenta en GitHub
- Cuenta en Netlify

## Paso a paso (resumen)
1. Subí el contenido de esta carpeta a un repo de GitHub.
2. En Netlify: **Add new site → Import from GitHub**.
3. En Netlify: **Site settings → Identity → Enable Identity**.
4. En Netlify: **Identity → Services → Enable Git Gateway**.
5. Entrá a: `https://TU-SITIO.netlify.app/admin` y creá tu usuario.

## Importante
En `public/admin/config.yml` reemplazá:
`repo: TU_USUARIO/TU_REPO`
por tu usuario y repo reales.
